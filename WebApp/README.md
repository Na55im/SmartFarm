## Installation

* Requirements
	* NodeJS &amp; npm
	* Angular-CLI (v1.0.0)
		* npm install -g @angular/cli@1.0.0
* Dependencies und Source Files
	* <pre>npm install</pre>
    * <pre>grunt watch</pre>
	
## Starten der WebApp

* Starten des Servers (localhost)
    * <pre>ng serve</pre>
* Starten des Servers
    *<pre>ng serve --host 0.0.0.0</pre>

# Angular WebApp - Project SmartFarm

Author: Sebastian Schulz 

## Installation

* Requirements
	* NodeJS (beinhaltet npm)
	* Angular-CLI (v1.0.0)
* Dependencies und Source Files

## Getting started

```
npm install
```

```
npm install -g @angular/cli@1.0.0
```
	
```
grunt watch
```

## Start the WebApp

* starts the WebApp in localhost
```ng serve
```
* starts the WebApp
```
ng serve --host 0.0.0.0
```

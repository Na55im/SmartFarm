Einleitung

SmartFarm ist ein Projekt für das Open-Source Betriebssystem RIOT. Das Ziel ist die Erkennung und Visualisierung von Bauernhöfen mit Weidezäunen.

Server und Framework

Grundlage ist das JS Framework Angular2 JS. Hierfür wird NPM und NodeJS benötigt.

Für die weiteren Funktionen und Module wird weiterhin Bower sowie Grunt benutzt.
